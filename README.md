# 🎓 Lecture Video to Notes AI

This project converts lecture videos into summarized notes using AI.

## 🚀 Features
- Upload lecture video
- Converts speech to text
- Generates summarized notes
- Supports multiple languages

## 🧠 Technologies Used
- OpenAI Whisper
- Hugging Face Transformers
- Streamlit

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python -m streamlit run app.py

## 🎯 Use Case
Helps students and professionals save time by converting long lectures into short notes.
